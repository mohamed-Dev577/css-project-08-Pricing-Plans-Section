# 💰 Pricing Plans Section

مشروع Front-End بسيط لعرض خطط الاشتراك باستخدام HTML و CSS.

يهدف المشروع إلى التدريب على:
- Flexbox
- Position
- Hover Effects
- Transform
- Transition
- تصميم احترافي لعرض الباقات

---

## 🚀 فكرة المشروع

تصميم قسم يحتوي على 3 خطط:

- الخطة المجانية
- الخطة الاحترافية (الأكثر طلباً)
- الخطة المتقدمة

مع إبراز الخطة الاحترافية بإطار مميز وشارة علوية.

---

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- Flexbox Layout
- Position (Relative & Absolute)
- Transform & Transition

## 🧠 ماذا تعلمت من المشروع؟

- كيفية تنظيم العناصر بجانب بعضها باستخدام Flexbox
- فهم الفرق بين relative و absolute
- تحريك العناصر باستخدام transform
- إضافة تأثيرات ناعمة باستخدام transition
- إبراز عنصر معين داخل التصميم
- إنشاء Pricing Section مشابه للمواقع الحقيقية
